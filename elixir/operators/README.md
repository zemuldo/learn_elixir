# Operators

## Operators

### Basic
Like any other language, Elixir prvides basic arithmetic operators:
```elixir
  Elixir|iex|16▶ 3 + 6
  9
  Elixir|iex|17▶ 7 * 7
  49
  Elixir|iex|18▶ 30 / 7
  4.285714285714286
  Elixir|iex|19▶ div 30, 7
  4
  Elixir|iex|20▶ rem 30, 7
  2
  Elixir|iex|21▶
```

### String Concat
```elixir
  Elixir|iex|21▶ "foo" <> "bar"
  "foobar"
  Elixir|iex|22▶ 
```
